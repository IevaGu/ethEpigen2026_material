Downloading the data

Downloading ATAC-seq peak counts in the hippocampus upon stress (a subset of the original data, already in SummarizedExperiment format). The data is from the mouse ensembl GRCm38 genome.

options(timeout=1800)

Reading peak file:

peaks <- readRDS("ATACseq_peaks.peakCounts.SE.rds")

Differential analysis with edgeR

Comparing stressed (denoted ‘FSS’ – forced swim stress) and control animals

Defining groups in the data set:

group <- ifelse(grepl("Control", colnames(peaks)), "Control", "FSS")
group <- factor(group)
colData(peaks)$group <- group

Giving the rows meaningful names:

rr <- rowRanges(peaks)
rownames(peaks) <- paste0(as.character(seqnames(rr)), ":", start(rr), "-", end(rr))

Performing differential analysis:

# Putting the data in the object structure used by edgeR:
dds_group <- edgeR::DGEList(assay(peaks), group=peaks$group)
# Normalizing using default TMM normalization:
dds_group <- edgeR::calcNormFactors(dds_group)
# When testing differences, it is not enough to know if means are different; we want to see if means are different relative to variability. That's why estimating dispersions is necessary: 
dds_group <- edgeR::estimateDisp(dds_group)
## Using classic mode.
# Performing differential analysis:
et_group <- as.data.frame(topTags(exactTest(dds_group), Inf))

Looks like there is one outlier (Control_female_GSM6041843.bam) that biases the differential analysis. However, it would not be a good idea to exclude the sample based solely on the heatmap. The proper approach would be, in order of priority:

  • Checking for corrections of ATAC-seq data, which is prone to technical biases.

  • If the sample is technically bad, then this should show in some metrics that are independent of the experimental design. If such metrics unambiguously show it as an outlier, we may exclude it.

  • Using surrogate variable analysis or some derivative of it.

Let’s check what happens if we exclude the sample just for the fun of it.
