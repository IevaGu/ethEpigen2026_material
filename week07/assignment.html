The output is a SummarizedExperiment object, where columns are the samples and the rows are the motifs. The values represent the relative accessibility of each motif in each sample. The motif deviations can be used to perform differential analysis.

Differential analysis

Here we identify motifs that change in accessibility between males and females.

dev$group <- ifelse(grepl("female", colnames(dev)), "Female", "Male")
design <- model.matrix(~group, data=as.data.frame(colData(dev)))
fit <- eBayes(lmFit(assay(dev, "z"), design))
res <- as.data.frame(topTable(fit, number = Inf))
## Removing intercept from test coefficients
res$motif <- row.names(res)
ggplot(res, aes(logFC, -log10(adj.P.Val), label=motif)) + geom_text()

Plotting a heatmap of the deviations for the top few candidate motifs.
